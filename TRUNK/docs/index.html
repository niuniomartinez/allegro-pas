Welcome to Allegro.pas

Allegro.pas is a wrapper to allow Pascal compilers (such as Free Pascal) to use the Allegro library to create computer games.

Who is Allegro.pas For?

Allegro.pas is right for you if:

  • You want to create games from the inside out.
  • You are a begginer, or amateur, or profesional game maker.
  • You want to use Pascal, with or without Lazarus, to create your games.
  • You like retro-style games, or modern ones.
  • You want clear documentation.

Where to start?

Read the README file, then read the Introduction section.

Also, you can peek the examples/ directory and the demonstration game at the demo/ directory to see real-life examples about how to use Allegro.

Contact info

The Allegro.pas project web site is at http://allegro-pas.sourceforge.net/.  There you can download the latest versions, as well as the full documentation.  Also you can read news and about projects that use Allegro.pas.

If you want to ask, comment or make suggestions about Allegro.pas you can do it by joining the Allegro.pas mailing list.  Also you can ask at the Pascal Game Development forums.
