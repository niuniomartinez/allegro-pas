============
Introducción
============

  Allegro.pas es una envolvente para poder utilizar la biblioteca Allegro con
  compiladores Pascal, como Free Pascal y Delphi.

  En principio, la he programado para uso propio, por lo que no he incluido
  toda la funcionalidad disponible en Allegro.  Aun así,  la iré corrigiendo y
  ampliándo según vaya necesitándose.  También administraré las modificaciones
  o añadidos que lleguen a mis manos para que estén disponibles para todos.

============================
Instrucciones de instalación
============================

  Actualmente Allegro.pas sólo funciona con Windows, ya que no tengo mucha
  experiencia trabajando con "shared objects" en sistemas *NIX.  No he escrito
  un guión de instalación (de nuevo me falta experiencia), pero puede hacerse a
  mano fácilmente.

  Lo más fácil es descargarse la versión binaria para Windows, ya que contiene
  todos los archivos necesarios y no tendrá compilar la librería dinámica.  El
  paquete con el código fuente está pensado para programadores expertos y para
  aquellos que colaboren en el desarrollo de Allegro.pas y es explicado más
  adelante.


Versión binaria:

  Una vez descargada la versión binaria, descomprima el archivo en su disco
  duro.  Abra el directorio que se creará con el explorador y busque el
  directorio "lib".  Ahí encontrará dos archivos llamados "alleg42.dll" y
  "alpas42.dll", copielos en  el directorio del sistema (normalmente
  C:\WINDOWS\SYSTEM32\).  Tenga en cuenta que es posible que necesite permisos
  especiales para hacerlo.

  Después copie el directorio "lib" en el directorio de bibliotecas de su
  compilador y configure este para que pueda encontrarlo cuando lo necesite:

  * Si utiliza Delphi, añada este en el "SearchPath" del proyecto.  Esto es,
    acceda, mediante el menú, a
   "Project>Options>Directories/Conditionals>Search Path" y añada la ruta.

  * Si utiliza Lazarus, acceda al menú "Proyecto>Opciones del compilador..." y
    añada el directorio en el campo "Other Unit Files (-Fu)".

  * Si utiliza Free Pascal, indique al compilador la ruta del directorio
    mediante la opción "-Fu".

  Ya está disponible para utilizar.


Versión de código fuente:

  Antes de empezar, asegurese de que tiene instalados correctamente un
  compilador de C o de C++ (el más recomendable es MinGW32), el compilador
  Free Pascal y la biblioteca Allegro (http://alleg.sourceforge.net/).

  Después descargue el paquete con el código fuente y descomprimalo en su
  disco duro.

  Abra una consola (por ejemplo, "Inicio>Ejecutar...>cmd.com"), acceda al
  directorio raiz de Allegro.pas y utilice el guión "fix.bat" ó "fix.sh",
  dependiendo de su entorno;  al ejecutar el guión sin parámetros este le
  devolverá la lista de los parámetros disponibles.  Por ejemplo:

  ~# fix
  >Usage: fix platform
  >
  >Where platform is one of: djgpp, mingw32 or linux.
  >
  >NOTICE:
  >You can also call: fix test
  >to check if your system can compile this project.
  >
  >
  >Done!
  >
  ~# fix mingw32
  >Configuring for Windows (Mingw32)...
  >Done!

  Una vez esté preparado, ejecute en ese mismo lugar "make".  NOTA:  Está
  preparado para utilizar la utilidad make de GNU, no la utilidad fpmake.  Si
  ejecuta "make help" le mostrará una lista de las opciones disponibles, siendo
  la más adecuada "make all".  Por ejemplo:

  ~# make help
  >Allegro.pas - http://allegro-pas.sourceforge.net/
  >--------
  >- Help -
  >--------
  >make all            } Builds the whole project.
  >make lib            } Creates the library only.
  >make examples       } Builds the examples only.
  >make tools          } Builds the tools only.
  >make demo           } Builds the demo game.
  >make clean          } Deletes temporary files.
  >make veryclean      } Deletes all builded files.
  >make status         } Shows some configuration variables.
  >make test           } Check program.
  >make help           } Shows this help.
  ~# make all
  >Allegro.pas - http://allegro-pas.sourceforge.net/
  > ...
  >Finished Allegro.pas
  >To create the HTML documentation, run 'make' from the docs directory

  Una vez finalizada la compilación (deberá aparecer el mensaje "Finished
  Allegro.pas") continúe la instalación siguiendo las indicaciones de la
  versión binaria, descritas más arriba.



======================
Utilizar la biblioteca
======================

  Como habrá visto, existe un buen surtido de unidades.  Lo he hecho así para
  que fuera más fácil de mantener.

  La documentación debe generarse con la utilidad fpdoc utilizando el archivo
  "docs.xml" sito en el directorio "docs".  Si lo desea, puede descargar o
  consultar la última actualización en la propia web de Allegro.pas, en
  http://allegro-pas.sourceforge.net/.



========================================
Distribuir programas que usan Alegro.pas
========================================

  Si va a compartir sus programas escritos usando Allegro.pas con amigos o a
  través de la red, no olvide incluir los archivos alpas42.dll y alleg42.dll
  junto con los ejecutables y los archivos de datos, de lo contrario su
  programa no funcionará.



=====================
Se busca colaboración
=====================

  Como ya se ha dicho, esta versión está incompleta.  Si quiere, puede
  colaborar en su finalización y añadir su nombre a la lista de autores.
  Necesitamos desde simples probadores hasta personas que ayuden a completar el
  manual de programación, que lo traduzcan a otros idiomas o que escriban las
  envolventes que faltan, demostraciones y ejemplos de uso.

  Si desea colaborar, lea la siguiente sección y póngase en contacto *antes* de
  empezar a trabajar.



=======================
Información de contacto
=======================

  Encontrará la información referente a Allegro.pas y podrá encontrar las
  últimas versiones liberadas en http://allegro-pas.sourceforge.net.  El sitio
  del proyecto está en http://sourceforge.net/projects/allegro-pas/.
  Allí encontrará una lista de correo y un foro público donde podrá exponer sus
  dudas y hacer sugerencias.

  Otro foro interesante sobre la programación de juegos en Pascal, que también
  puede ayudarle si tiene problemas con Allegro.pas, se encuentra en el sitio
  "Pascal Game Development" (http://www.pascalgamedevelopment.com/).

  Si quiere obtener el máximo provecho, le recomendamos que siga las siguientes
  indicaciones:

    * Antes de hacer cualquier pregunta sobre la biblioteca lea la
      documentación.  Si no encuentra la respuesta en ella o no entiende algo,
      entonces sí, diríjase al foro.  Si cree que falta algo en los documentos,
      no se explica correctamente o que hay errores, lo sabemos: ya dijimos que
      estamos trabajando en ello, ¿verdad?
    * Antes de enviar un mensaje compruebe si ya se ha comentado con
      anterioridad su duda o propuesta.  El foro dispone de un buscador que
      facilita la tarea de comprobación.  Recuerde que el repetir las mismas
      preguntas, aunque sea en distintos foros, no asegura encontrar mejores
      o más rápidas respuestas.
    * Ponga en el asunto una frase que resuma su mensaje.  Frases como
      "Necesito ayuda" o "No funciona" tal vez no atraigan la atención de la
      gente que pueda ayudarle.  Utilize frases como "No se muestra el sprite
      donde quiero" o "No se reproduce el sonido correctamente".
    * Si la consulta se refiere a un programa que ha escrito usted, procure
      incluir el código donde crea que está el problema.  No es preciso que
      incluya todo el programa (a menos que alguien se lo pida expresamente).

  Siguiendo estas sencillas normas tendrá más posibilidades de obtener una
  respuesta rápida, correcta y adecuada a sus preguntas y sugerencias.
